Text Analyzer Project (Python)
📌 Description

The Text Analyzer is a Python-based mini project that analyzes user input text. It performs character-level and word-level analysis including counting vowels, consonants, uppercase and lowercase letters, and word frequency. It also displays the top 3 most frequent words using dictionary operations.

🚀 Features
Counts vowels and consonants
Counts uppercase and lowercase letters
Splits and analyzes words
Word frequency calculation using dictionary
Displays top 3 frequent words
Clean function-based structure
🧾 How to Run
python text_analyzer.py
📥 Example Input
Python is easy python is fun
📊 Example Output
Vowels: 8  
Consonants: 12  
Uppercase Letters: 1  
Lowercase Letters: 18  

Top 3 Frequent Words:
python : 2
is : 2
easy : 1
